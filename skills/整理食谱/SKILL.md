# 整理食谱

从小红书链接中提取食谱内容，整理成标准格式保存到 Obsidian 食谱库。

## 基本信息

- **Obsidian vault 路径**：`E:\Obsidian\ob库\电脑仓库`
- **食谱保存目录**：`Atlas-Knowledge/厨艺/`
- **食谱索引文件**：`Atlas-Knowledge/厨艺/0 食谱.md`

## 执行流程

### 第一步：获取页面内容

使用 `mcp__claude-in-chrome__tabs_context_mcp` 获取当前标签页，然后为每个链接创建新标签页（`mcp__claude-in-chrome__tabs_create_mcp`）并导航（`mcp__claude-in-chrome__navigate`）。

- 优先使用 `mcp__claude-in-chrome__get_page_text` 获取正文
- 如果正文内容不足，使用 `mcp__claude-in-chrome__read_page` 获取完整页面结构（depth=5）
- 注意：小红书视频笔记的详细步骤有时在图片中，需结合评论区和页面文本推断

### 第二步：分析与整合

- 判断每个链接是**食谱**还是**烹饪技巧**（如去骨、切法等技巧视频不单独成笔记，而是作为相关食谱的步骤或贴士补充）
- 提取：菜名、食材+用量、制作步骤、时间、份量、小贴士
- 多个链接可能共同支撑同一道菜（如技巧视频 + 食谱视频）

### 第三步：创建笔记文件

使用 `Write` 工具在 `E:\Obsidian\ob库\电脑仓库\Atlas-Knowledge\厨艺\[菜名].md` 创建文件，格式如下：

```markdown
# 菜名：[菜名]

> **描述**：[一句话描述，突出特色]
> **准备时间**：[X分钟] | **烹饪时间**：[X分钟] | **份量**：[X人份]

## 食材清单
- [ ] [食材]：[用量]
- [ ] [食材]：[用量]

## 制作步骤
1. **[步骤名]**：[具体操作]
2. **[步骤名]**：[具体操作]

## 小贴士
- [关键注意事项或技巧]

## 来源
- [视频标题](小红书链接)
```

### 第四步：更新索引

先用 `Read` 读取 `E:\Obsidian\ob库\电脑仓库\Atlas-Knowledge\厨艺\0 食谱.md`，再用 `Edit` 工具在末尾追加新菜名的 wikilink：

```markdown
## [[新菜名]]
```

## 注意事项

- 食材用量若视频未明确提及，根据常识给出合理估计并在贴士中注明"可根据口味调整"
- 步骤名加粗，使每步一目了然
- 来源部分保留原始小红书链接，方便日后回看视频
- 若有多个视频支撑同一道菜，来源部分列出所有相关链接
